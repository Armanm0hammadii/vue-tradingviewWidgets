# vue-tradingview-widgets

## Instalation

```bash
npm i vue-tradingview-widgets
```

## Import in component

```js
<template>
  <Chart />
  <Snaps />
</template>

<script lang="ts">
import { Chart, Snaps } from 'vue-tradingview-widgets';

export default {
  components: { Chart, Snaps },
};
</script>
```
